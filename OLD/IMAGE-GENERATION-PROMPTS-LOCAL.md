## IMAGE SPECIFICATIONS

### Technical Requirements
- **Resolution:** 300 DPI minimum for print, 72 DPI for web
- **Format:** PNG with transparency where appropriate, JPG for photos
- **Color Space:** sRGB for web, CMYK for print
- **Aspect Ratios:**
  - Equipment photos: 4:3 or 1:1 (square)
  - Coverage diagrams: 16:9 or 4:3
  - Hero shots: 16:9 widescreen
  - Charts/infographics: Variable based on content

### Style Guidelines
- **Equipment Photos:** Clean white/gray gradient backgrounds, professional studio lighting
- **Coverage Diagrams:** Technical/engineering style, clean lines, professional color coding
- **Setup Photos:** Real-world venue settings, professional installation appearance
- **Charts:** Modern, clean design matching website color scheme (blues, grays)

### Branding Notes
- Include QSC, Yamaha, Shure, DBX logos accurately where products are shown
- Avoid competitor branding in images
- Match website color scheme: #4299e1 (blue), #38a169 (green), #805ad5 (purple)
