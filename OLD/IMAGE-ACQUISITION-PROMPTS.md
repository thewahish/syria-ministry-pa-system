## Technical Specifications for All Assets

- **Resolution:** Minimum 4K (3840x2160) for hero images
- **Format:** PNG with transparency for diagrams, JPG for photography
- **Color Space:** sRGB for web, Adobe RGB for print
- **File Size:** Optimized for web delivery (<2MB per image)
- **Accessibility:** Alt text ready, high contrast ratios
- **Responsive:** Multiple breakpoints (mobile, tablet, desktop)

## Usage Rights and Requirements

- All images must be suitable for commercial use
- No visible competitor logos or trademarks
- Professional quality suitable for government presentations
- Culturally appropriate for Middle Eastern/Syrian context
- Bilingual caption ready (English/Arabic)

These prompts should generate a comprehensive visual library that supports the professional presentation of this PA system project while maintaining the sophisticated, government-appropriate aesthetic required for the Syria Ministry installation.
